✨ How This Site Was Built

I started with a simple idea: create a space that feels like a quiet dream. Every color, animation, and line of code was chosen with care.

💻 Tech Used

  • HTML5 for structure
  • CSS3 for styling, animations, and responsive design
  • JavaScript for interactivity (typing effect, messages, canvas stars)

🧠 Challenges I Faced

  • Making the navigation work on all pages
  • Balancing aesthetics with accessibility
  • Creating a consistent dreamy color palette

🎨 My Favorite Part

The floating character and the glowing ring — because it feels like magic ✨
